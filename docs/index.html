State of the State Book 2025

The Constitution of New York State requires the Governor to deliver an annual message to the Legislature regarding the state of the state. The State of the State proposals are the first step in defining the Governor's agenda in 2025.

The New York State Digital Service is working on creating a re-usable library of tools to more easily convert documents, PDFs, and data tables to simple and more accessible formats using semantic HTML and basic CSS. This site is evolving and a work in progress! If you have questions please feel free to contact us at Digital.Services@ogs.ny.gov
